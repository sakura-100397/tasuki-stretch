# タスキストレッチ 60分体験無料チケット 作成プロンプト

## 目的

タスキストレッチの60分体験無料チケットを、表面・裏面の2枚のPNG画像として作成するためのプロンプトです。  
参照画像の上品な美容サロン感、ピンクと深緑の配色、チケット風の外枠装飾を保ちながら、文字・装飾・画像の重なりやズレをなくして再構成します。

- 表面参照画像: `tasuki-60min-free-ticket-front-v1.png`
- 裏面参照画像: `tasuki-60min-free-ticket-back-v2.png`
- 出力サイズ: `1612px x 976px`
- 出力形式: PNG
- 保存先: `tasuki.ibjapan.jp/dev/images`

## 最重要ルール

1. 英字のブランド表記は入れない。ブランド名は日本語の「タスキ ストレッチ」のみ使用する。
2. 文字用カードや文字エリアの背景は不透明な白ベタにしない。背景画像が透けて見える半透明または透明の状態にする。
3. 文字、リボン、枠、QRカード、写真などのレイアウトオブジェクトは、中央揃え・左右揃えを明確にする。
4. 文字と写真、文字と枠、写真と区切り線、QRと装飾線が重ならないように十分な余白を取る。
5. 写真やQRコードを配置するときは、縦横比を変えて引き延ばしたり縮めたりしない。表示範囲をトリミングして調整する。
6. QRコードはAIで描かず、実際に読み取れる既存QR画像をそのまま配置する。
7. 裏面に体験内容を説明する冒頭セクションは作らない。関連する3項目の説明も入れない。
8. 日本語テキストは下記の正確な文言を使い、文字化けや余計な文言を入れない。
9. 医療行為、治療、痩身効果を保証する表現は入れない。

## 共通デザイン仕様

```text
1612px x 976pxの横長チケット画像。
白から淡いピンクのやわらかい背景に、右側または中央に明るい女性ストレッチ写真を配置する。
外周には細いコーラルピンクの二重飾り罫を入れる。
四隅はクラシックな招待状のような曲線装飾にする。
左右端には茶色の縦破線の切り取り線と小さなはさみアイコンを配置する。
アクセントカラーはコーラルピンク、深緑、オリーブ、淡いピンク。
文字は背景画像の上に直接配置するか、必要な場合のみ透明度のある淡いパネルを使う。
文字用パネルを使う場合も背景画像が見える透明感を残す。
```

### 推奨カラー

- 深緑: `#123D22`
- コーラルピンク: `#EF6464`
- 淡ピンク: `#FFF4F2`
- オリーブ: `#8A865E`
- 文字黒: `#1E1713`
- 破線ブラウン: `#6D4B3E`

### レイアウトの整列ルール

```text
表面:
左側の文字ブロックは同じ左端に揃える。
メイン見出し、区切り線、訴求文、リボン、最下部案内文は同じ中心線に揃える。
右側の人物写真は文字ブロックに重ならない位置に配置する。
英字ロゴが入っていた領域には、日本語ブランド名だけを中央配置する。

裏面:
左側は「おすすめ枠」「チェックリスト」「限定案内帯」「Access」を同じ左端に揃える。
チェックアイコンと各行テキストの縦位置を揃える。
右側はCTA見出し、QRカード、スマホ読み取り文言を同じ中心線に揃える。
左エリアと右エリアの間には縦破線を置き、どの要素も破線に重ならない。
写真は左エリア上部の余白に配置し、チェックリストや区切り線と重ならない。
```

### 画像配置ルール

```text
人物写真は縦横比を維持して配置する。
必要なサイズに合わせるときは、画像を引き伸ばさず、object-fit: cover のように表示範囲をトリミングする。
QRコードは正方形のまま配置し、縦横比を変えない。
背景写真の上に文字を置く場合は、文字の可読性を保つために薄い半透明グラデーションを重ねる。
ただし、文字用カードを白ベタで塗りつぶさない。
```

## 表面再現プロンプト

```text
タスキストレッチの60分体験無料チケット表面を作成する。
キャンバスは1612px x 976px。
全体は白から淡いピンクの清潔感ある背景。

写真は右側に収め、左側の文字ブロックに重ならない。

外周にはコーラルピンクの二重飾り罫、曲線の角飾り、左右の茶色い切り取り破線、小さなはさみアイコンを入れる。

上部中央には日本語で「タスキ ストレッチ」を配置する。
英字ブランド表記は入れない。
ブランド名の左右に小さなオリーブ色の葉装飾を置く。
ブランド名の下に細いコーラルピンクの水平線と小さなリボン装飾を入れる。

左側のメイン文字ブロックは、左端と中心線を揃えて配置する。
最も大きく「1回 8,800円の体験レッスン」を配置し、「8,800円」をコーラルピンクで強調する。
その下に「無料チケット」を大きく配置し、「無料」をコーラルピンク、「チケット」を深緑で配置する。
見出し下にコーラルピンクの点線を水平に入れ、中央に小さなリボン装飾を置く。

その下に「「姿勢」と「肌ツヤ」を速攻改善」をコーラルピンクで配置する。
さらに下に「ヨガでもピラティスでもない、新しい健康と美容の習慣」を黒または深緑で配置する。

下部中央にコーラルピンクのリボン帯を配置し、帯の中央に白文字で「初めての方限定」と入れる。
リボン帯の下に「ご予約時にこのチケットをご提示ください」を配置する。
すべての文字と装飾は重ならないよう、上下に十分な余白を取る。
文字用の背景カードを置く場合は半透明にし、背景画像が透ける状態にする。
```

## 裏面再現プロンプト

```text
タスキストレッチの60分体験無料チケット裏面を作成する。

関連する3項目の箇条書きも入れない。

左上に深緑の細い角丸枠を置き、枠内中央に「こんな方におすすめ」を配置する。
その下にチェックアイコン付きで以下4行を配置する。
チェックアイコンの中心と各行テキストの縦中央を揃える。
各行の左端を揃え、行間を均等にする。

写真は縦横比を維持し、表示範囲のトリミングで調整する。
写真はおすすめ枠、チェックリスト、縦破線と重ならない位置に置く。

左中央下に淡いピンクの半透明の限定案内帯を配置する。
```
